:. Gym Fee Tracker
===============

A full-stack web application for local gyms to manage member payments digitally with automatic monthly updates and real-time unpaid status tracking.

Features
--------

*   Member registration & management
    
*   Automatic monthly fee status updates
    
*   Real-time unpaid status tracking (e.g., “2 months unpaid”)
    
*   Mark as Paid / Delete / Undo functionality
    
*   Secure JWT authentication
    
*   Google OAuth login integration
    
*   Responsive dashboard for gym owners
    
*   Persistent cloud database storage with MongoDB
    

Tech Stack
----------

*   **Frontend:** React.js
    
*   **Backend:** Node.js, Express.js
    
*   **Database:** MongoDB
    
*   **Authentication:** JWT, Google OAuth
    
*   **APIs:** RESTful APIs
    

How It Works
------------

*   Gym owners can register and manage members digitally
    
*   The system automatically updates payment status every month
    
*   Members who miss payments are marked as unpaid in real time
    
*   Owners can instantly track dues, mark payments, delete records, or undo actions
    
*   Secure authentication ensures protected access to gym data
